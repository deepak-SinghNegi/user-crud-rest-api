# user-crud-rest-api


A full-stack practice project built to strengthen backend fundamentals and understand REST API development using Spring Boot and MongoDB.

## 🎯 Purpose of This Project

This project was built as a hands-on practice application to:

- Implement CRUD operations using Spring Boot
- Work with MongoDB (NoSQL database)
- Build RESTful APIs
- Connect frontend and backend
- Deploy a full-stack application on AWS
- Practice Git version control and project structuring
